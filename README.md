# True Trace Forge (AMMAM)

**Algorithmic Memory Mapping for Agentic Malware (AMMAM)**

True Trace Forge is an advanced cybersecurity dashboard and forensic analysis platform. It is designed to assist law enforcement, digital forensics analysts, and cybersecurity teams in instantly analyzing, tracking, and mitigating AI-generated media threats such as Deepfakes, voice clones, and algorithmic malware.

### Key Features
- **Police Command Center:** A real-time triage dashboard that categorizes threats into Critical, Suspicious, or Authentic states for rapid response.
- **Deepfake Analyzer:** Upload and instantly analyze media for face swaps, synthetic artifacts, and voice anomalies.
- **Forensic Lab:** Deep-dive analysis tools including Manipulation Heatmaps, Spectrograms, Temporal Scanning, and Error Level Analysis (ELA).
- **Propagation Tracking:** Monitor the spread of malicious media across global networks and trace the chain of custody.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/true-trace-forge-main.git
   cd true-trace-forge-main
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Run the application instance:**
   ```bash
   npm run dev
   ```

The application will launch on your local network. Simply click the `localhost` link provided in your terminal (typically `http://localhost:8080/` or `http://localhost:5173/`) to access the True Trace Forge command center.
